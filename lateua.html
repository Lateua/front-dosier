El proyecto se desarrolla a partir de las tres claves conceptuales. Tus referencias junto con el estudio de las piezas claves que estas contienen nos llevan a definir un estilo. Con ello se genera un entorno estético para llevar a cabo el proyecto decorativo. Se define ecléctico, moderno, que permite jugar con materiales y texturas, creando de esta manera un espacio con personalidad y carácter.
Colores Mood
Son colores que transmiten sosiego, elegancia y sofisticación. Aportarán toques de color en detalles dentro de espacios mas neutros: cojines, plaids, butaca/puff. Se pueden utilizar en todas las estancias y permiten unificar ambientes cuando se establece una transición suave entre los distintos tonos
